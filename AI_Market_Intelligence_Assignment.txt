================================================================
  ASSIGNMENT: AI-POWERED MARKET INTELLIGENCE & TREND ANALYTICS PLATFORM
================================================================

OBJECTIVE
---------
Build a production-grade backend system that:
- Ingests real-world data from a third-party API
- Processes and stores it in MongoDB
- Performs analytics
- Exposes insights through a FastAPI-based REST API
- Must be deployed and publicly accessible

PIPELINE FLOW
-------------
Third-Party API --> Data Ingestion --> MongoDB --> Data Processing --> FastAPI --> Deployment


================================================================
THIRD-PARTY API (Choose ONE)
================================================================
1. News API (Recommended)
2. Alpha Vantage (Stocks/Crypto)
3. Adzuna API (Job Market)


================================================================
FUNCTIONAL REQUIREMENTS
================================================================

1. DATA INGESTION
-----------------
- Fetch data from the selected API
- Normalize and clean data
- Store raw data in MongoDB (raw_data collection)

2. DATA PROCESSING
------------------
- Transform raw data into structured format
- Extract:
    * Keywords
    * Sentiment (basic rule-based or API-based)
    * Trend score
- Store processed data in (processed_data collection)

3. DATABASE DESIGN (MongoDB)
----------------------------

Raw Collection Example:
{
    "title": "...",
    "source": "...",
    "published_at": "...",
    "content": "...",
    "category": "business"
}

Processed Collection Example:
{
    "title": "...",
    "sentiment": "positive/negative",
    "keywords": ["AI", "economy"],
    "score": 0.87,
    "trend_score": 12
}


================================================================
DATA ANALYSIS REQUIREMENTS
================================================================

A. TREND DETECTION
- Identify top keywords (last 24h / 7 days)

B. SENTIMENT ANALYSIS
- Classify data into: positive, negative, neutral

C. TIME-SERIES ANALYSIS
- Count items per day
- Identify growth trends


================================================================
FASTAPI REQUIREMENTS
================================================================

ENDPOINTS:
----------
POST  /ingest              --> Fetch and store data
GET   /trends?days=7       --> Return trending keywords
GET   /insights            --> Aggregated insights
GET   /search?q=keyword    --> Search functionality
GET   /analytics/summary   --> Summary statistics

MANDATORY FEATURES:
-------------------
- Pydantic models
- Query parameters
- Pagination
- Error handling
